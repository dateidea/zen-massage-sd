# Image Brief — Feel Good Spa

Every image referenced in the build lives at `/public/images/<id>.jpg`.
Generate them in **ChatGPT (GPT-4o image generation)**, one at a time, in
the order below — sharing one conversation thread is fine and keeps the
visual language consistent.

**Shared style language to reuse across every prompt:**

> Photorealistic editorial photograph for a premium massage studio website.
> Lighting: natural, soft, directional, late-afternoon golden warmth, slight
> film grain, gentle shallow depth of field. Color palette: muted, desaturated,
> warm neutrals — off-white (#F7F3EC), warm charcoal (#1C1A17), terracotta
> clay accent (#B8754A). No oversaturation, no HDR look, no AI-glossy
> rendering. No text, no logos, no watermarks. People (when present) look
> real — pores, asymmetry, natural skin tones, ethnically varied, candid
> expressions. Hands and faces must be anatomically correct or out of frame.
> Shot on a 35mm or 50mm full-frame lens.

## After generation — file processing

1. Save each output as `<id>.jpg` into `/public/images/`.
2. Convert to AVIF + WebP (e.g. `npx @squoosh/cli --avif auto --webp auto public/images/*.jpg`).
3. Inspect every image for AI failure points: hands, ears, jewelry, text,
   weird symmetry. Regenerate if any are wrong. Do not ship cursed images.
4. The final visual language across all 10 images should look like one
   coherent shoot — same lighting direction, same grain, same warm color
   treatment. If image 5 looks like Midjourney v3 next to image 1, redo it.
